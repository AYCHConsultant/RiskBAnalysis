Community

Who

MONARC is lead by the CASES team. CASES is also organizing the MONARC trainings.

The full list of contributors is available on GitHub.

If you have any questions about MONARC don’t hesitate to contact the team at info@cases.lu.

The data

We are working on a solution which will let the users share their MONARC JSON objects much easier:
http://objects.monarc.lu

The source code of the prototype is available here:
https://github.com/CASES-LU/MOSP

MONARC objects (threats, assets, vulnerabilities) are licensed under CC0 1.0 Universal (CC0 1.0) - Public Domain Dedication. If a specific author wants to license an object under a different license, a pull request can be requested.
