Installing the demo Virtual Machine

Download the image

Download the last released image, MONARC_demo.ova

Importing the image

You will need a tool for virtualization which accepts .ova files. In this guide, VirtualBox will be used.

Import the image in your application by either just double-click on the .ova file (if VirtualBox is associated with .ova files) or launch VirtualBox and click on File -> Import appliance.

Configuration of VirtualBox

Now, set-up a new “host only network” (file -> preferences -> network)

After this, fill out using the following parameters :

You also have to modify the network interface of the virtual machine.

Just be sure that “host only adapter” is selected and the name is the one just created.

NOTE: The box “cable connected” has to be ticked.

Run the server

Now that the image is installed and configured, you can run the server by selecting “demo_monarc” and clicking start.
